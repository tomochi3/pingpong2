# Ping Pong

昔ながらのピンポンをベースに、加速、溜めショット、スピン、カーブ、同じWi-Fiでの対戦を入れたWebゲームです。

## 起動方法

```bash
npm start
```

起動すると、ターミナルに次のようなURLが表示されます。

```text
Ping Pong LAN server listening on http://127.0.0.1:5173/
LAN URL: http://192.168.x.x:5173/
```

同じPCで遊ぶ場合は `http://127.0.0.1:5173/` を開きます。

同じWi-Fiの別端末と対戦する場合は、表示された `LAN URL` を相手に開いてもらいます。1人目が左側、2人目が右側を操作します。

## 基本ルール

- 5回ゴールが入るとゲーム終了です。
- 点数はゴール点とスピン点で増えます。
- ボールはラリーが続くほど少しずつ速くなります。
- パドルの中心に近いほど速く返球できます。
- ゴール後、次のサーブが始まるまでの間もパドル操作できます。

## キーボード操作

### 1人プレイ / LAN対戦

- `W` / `↑`: 上方向に加速
- `S` / `↓`: 下方向に加速
- `←` / `A`: 溜めショット
- `Space`: 開始 / 再戦
- `P` / `Enter`: 一時停止
- `R`: リスタート
- `E`: 簡単モード ON/OFF（初期ON）
- `T`: 通常 / トレーニング / ハードを切替
- `H`: ハードトレーニングに切替
- `F`: フルスクリーン

`←` / `A` は長押しで溜め、離すとショット準備状態になります。溜めている間だけボールはパドルをすり抜けます。溜めていない時は通常返球でき、ショット準備中は溜めた分だけ強く返せます。

簡単モードは最初からONです。自分のパドル中心がボールのY位置に自動で揃います。通常時は必ず返球できますが、溜めっぱなしの時はヒットできません。溜め中に上下加速すると回転も貯まり、溜めを離してから当てるとスピン/カーブに反映されます。難しくしたい時は `E` でOFFにできます。

トレーニングモードでは、AI相手が必ず返球します。ハードトレーニングモードでは、AI相手がより速く、回転のかかった返球をしてきます。

### 同じPCで2人対戦

メニュー画面で `2` を押すと、同じPCで2人対戦が始まります。

- 左プレイヤー: `W` / `S` で上下移動、`A` で溜めショット
- 右プレイヤー: `↑` / `↓` で上下移動、`←` で溜めショット
- `1`: 1人プレイで開始
- `2`: 同じPCの2人対戦で開始
- `R`: 現在のモードで再戦

## ドラッグ操作

ボタンなしでもドラッグだけで遊べます。

- 上へドラッグ: 上方向に加速
- 下へドラッグ: 下方向に加速
- 左へドラッグ: 溜めショット
- 左へ引いた状態から戻す / 指を離す: ショット解放

スマホやタブレットでは、画面を引っ張る感覚で操作できます。

## 解放モード

このゲームは最初から解放モードです。

上下を同時に入力すると、上下両方の速度が溜まります。片方を離すと、残った方向の速度で一気に動きます。

例: 上下を同時に溜めたあと、上入力だけを離すと、下方向へ猛スピードで移動します。

## スピンとカーブ

溜め中に上下へ加速するほど、返球時に強いスピンがかかります。

- スピン量に応じて小数点2桁のスピン点が入ります。
- スピンが強いほどボールは大きく弧を描きます。
- 強いスピンの時は音も変わります。
- スピン点に上限はありません。

## 溜めショットの点数

溜めショットで自分がボールを速くした場合、その速さは自分がゴールした時のボーナスになります。

溜めている間だけボールはパドルをすり抜けます。簡単モードでも、溜めっぱなしの時はヒットできません。溜めていない時は通常返球でき、溜めを離した直後は強い返球になります。

ただし、相手がそのボールでゴールした場合は、自分が速くした分のボーナス点は相手に入りません。

## LAN対戦

1. ホスト側のPCで `npm start` を実行します。
2. ホストは `http://127.0.0.1:5173/` を開きます。
3. 同じWi-Fiにいる相手は、ターミナルに表示された `LAN URL` を開きます。
4. 画面左下に `LAN LEFT / 対戦中`、`LAN RIGHT / 対戦中` と表示されれば接続成功です。

3人目以降が開いた場合は観戦扱いになります。

## 同じPCで2人対戦

1. ゲーム画面を開きます。
2. メニュー画面で `2` を押します。
3. 左プレイヤーは `W/S/A`、右プレイヤーは `↑/↓/←` で操作します。
4. 画面左下に `LOCAL 2P / 同じPCで対戦中` と表示されれば、同じPC対戦中です。

## 一人で遊ぶ場合

LANサーバーにつながっていない状態では、自動でAI相手の一人プレイになります。
