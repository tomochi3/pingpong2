# Ping Pong

昔ながらのピンポンをベースに、加速、溜めショット、スピン、カーブ、同じWi-Fiでの対戦を入れたWebゲームです。

## 起動方法

```bash
npm start
```

起動すると、ターミナルに次のようなURLが表示されます。

```text
Ping Pong LAN server listening on http://127.0.0.1:5173/
LAN URL: http://192.168.x.x:5173/
```

同じPCで遊ぶ場合は `http://127.0.0.1:5173/` を開きます。

同じWi-Fiの別端末と対戦する場合は、表示された `LAN URL` を相手に開いてもらいます。1人目が左側、2人目が右側を操作します。

## 基本ルール

- 5回ゴールが入るとゲーム終了です。
- 点数はゴール点とスピン点で増えます。
- ボールはラリーが続くほど少しずつ速くなります。
- パドルの中心に近いほど速く返球できます。
- ゴール後、次のサーブが始まるまでの間もパドル操作できます。

## キーボード操作

- `W` / `↑`: 上方向に加速
- `S` / `↓`: 下方向に加速
- `←` / `A`: 溜めショット
- `Space`: 開始 / 再戦
- `P` / `Enter`: 一時停止
- `R`: リスタート
- `F`: フルスクリーン

`←` / `A` は長押しで溜め、ボールに当たる直前に離すとショット準備状態になります。タイミングよく離さないとボールに当たりません。

## ドラッグ操作

ボタンなしでもドラッグだけで遊べます。

- 上へドラッグ: 上方向に加速
- 下へドラッグ: 下方向に加速
- 左へドラッグ: 溜めショット
- 左へ引いた状態から戻す / 指を離す: ショット解放

スマホやタブレットでは、画面を引っ張る感覚で操作できます。

## 解放モード

このゲームは最初から解放モードです。

上下を同時に入力すると、上下両方の速度が溜まります。片方を離すと、残った方向の速度で一気に動きます。

例: 上下を同時に溜めたあと、上入力だけを離すと、下方向へ猛スピードで移動します。

## スピンとカーブ

パドルが速く動いているほど、返球時に強いスピンがかかります。

- スピン量に応じて小数点2桁のスピン点が入ります。
- スピンが強いほどボールは大きく弧を描きます。
- 強いスピンの時は音も変わります。
- スピン点に上限はありません。

## 溜めショットの点数

溜めショットで自分がボールを速くした場合、その速さは自分がゴールした時のボーナスになります。

ただし、相手がそのボールでゴールした場合は、自分が速くした分のボーナス点は相手に入りません。

## LAN対戦

1. ホスト側のPCで `npm start` を実行します。
2. ホストは `http://127.0.0.1:5173/` を開きます。
3. 同じWi-Fiにいる相手は、ターミナルに表示された `LAN URL` を開きます。
4. 画面左下に `LAN LEFT / 対戦中`、`LAN RIGHT / 対戦中` と表示されれば接続成功です。

3人目以降が開いた場合は観戦扱いになります。

## 一人で遊ぶ場合

LANサーバーにつながっていない状態では、自動でAI相手の一人プレイになります。
